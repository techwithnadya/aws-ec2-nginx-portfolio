# AWS EC2 Nginx Portfolio Website

This project demonstrates how to deploy a website using AWS EC2 and Nginx.

## Architecture

Domain 
↓ 
DNS 
↓ 
Elastic IP 
↓ 
EC2 Instance 
↓ 
Nginx Web Server 
↓ 
Website

## Technologies Used

- AWS EC2
- Nginx
- Linux
- DNS
- Let's Encrypt SSL

## Skills Learned

- SSH access to EC2
- Linux server basics
- Web server configuration
- DNS configuration
- HTTPS security
